142 // UART serial from CC2650 to TM4C123/MSP432
143 // UART serial from TM4C123/MSP432 to CC2650
144 // MRDY, an output from TM4C123/MSP432 and an input to CC2650
145 // SRDY, an input to TM4C123/MSP432 and an output from CC2650
146 // Reset, an output from TM4C123/MSP432 to hardware reset to CC2650
147 // MRDY is a GPIO pin whose logic level indicates the state of the master processor.
148 // MRDY is an active low (low true) signal.
149 // An MRDY event indicates that the master processor is ready to send or receive data.
150 // SRDY is a GPIO pin whose logic level indicates the state of the slave processor.
151 // SRDY is an active low (low true) signal.
152 // An SRDY event indicates that the slave processor is ready to send or receive data.
153 // see GPIO.c file for hardware connections
