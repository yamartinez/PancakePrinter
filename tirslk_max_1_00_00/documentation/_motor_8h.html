Detailed Description

Set motor speed and direction.

Provide mid-level functions that initialize ports and set motor speeds to move the robot.

◆ Motor_Init()

void Motor_Init ( void  )

Initialize motor interface.

Initialize GPIO pins for output, which will be used to control the direction of the motors and to enable or disable the drivers. The motors are initially stopped, the drivers are initially powered down, and the PWM speed control is uninitialized.

none
